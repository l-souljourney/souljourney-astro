# Design Document: v2.3 Legacy Cleanup & Bug Fixes

**Date:** 2026-02-15
**Status:** Draft / Pending Implementation
**Version:** v2.3

## 1. Overview
Version 2.3 focuses on "slimming down" the codebase by removing unused legacy components and scripts from the 1.x era, and fixing critical UI bugs discovered during the 2.x migration.

## 2. Goals
- **Minimalism:** Remove at least 5 unused client-side scripts and 1 component.
- **Reliability:** Fix the Table of Contents (TOC) highlighting and prevent script crashes when configuration is missing.
- **Type Safety:** Resolve remaining LSP/TypeScript warnings in core components.

## 3. Detailed Components

### 3.1 Legacy Cleanup
- **Scripts to Remove (`src/scripts/`)**: `HanAnalytics.ts`, `SeoPush.ts`, `ViewImage.ts`, `GoogleAd.ts`, `LivePhoto.ts`, `Search.ts`.
- **Assets to Remove (`public/assets/js/`)**: `view-image.min.js`, `livephotoskit.js`.
- **Components to Remove**: `src/components/GoogleAd/`.
- **Dependencies to Uninstall**: `less`, `@waline/client`, `@types/nprogress`.

### 3.2 Bug Fixes
- **TOC Highlight**: Fix the `.toc` vs `aside` selector mismatch in `TOC.astro`.
- **TypeWrite**: Add length checks to `TypeWriteList` in `TypeWrite.ts` to prevent `.substring` errors on `never[]`.
- **Config Cleanup**: Remove deprecated keys from `src/config.ts` to reduce cognitive load for developers.

## 4. Acceptance Criteria
- `pnpm build` completes without errors.
- Browser console is clean (no missing module errors).
- TOC correctly highlights headings during scroll.
- Website remains functional in both Light and Dark modes.
